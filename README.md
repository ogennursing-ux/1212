# תיק עובד זר — עוגן סיעוד

מערכת עצמאית לניהול תיקי עובדים זרים: פרטים אישיים, תוקף מסמכים (דרכון, אשרה,
היתר, ביטוח), העלאת סריקות, והפקת חוזה העסקה שנתי מתוך פרטי התיק.

המערכת עצמאית לחלוטין ואינה קשורה למערכות אחרות. הנתונים והסריקות נשמרים
מקומית בדפדפן (IndexedDB) — ללא שרת.

## הרצה מקומית

```bash
npm install
npm run dev
```

## בנייה ופרסום

```bash
npm run build      # יוצר את התיקייה dist/
```

הפרסום ל-GitHub Pages אוטומטי: כל דחיפה ל-`main` מפעילה את
`.github/workflows/deploy.yml`, שבונה ומפרסם את האתר.
